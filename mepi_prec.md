# Epipolar Geometry: Error Metrics

This document outlines the mathematical implementation of epipolar distance metrics used to evaluate matches between point pairs $(p_0, p_1)$ given an Essential/Fundamental matrix $E$.

## 1. Core Definitions

Given homogeneous points $\tilde{\mathbf{p}}_0 = (x_0, y_0, 1)^\top$ and $\tilde{\mathbf{p}}_1 = (x_1, y_1, 1)^\top$:

- **Algebraic Error ($e$):** $$e = \tilde{\mathbf{p}}_1^\top \mathbf{E} \tilde{\mathbf{p}}_0$$
- **Epipolar Line in Image 1:** $$\ell_1 = \mathbf{E}\tilde{\mathbf{p}}_0 = (a_1, b_1, c_1)^\top$$
- **Epipolar Line in Image 0:** $$\ell_0 = \mathbf{E}^\top \tilde{\mathbf{p}}_1 = (a_0, b_0, c_0)^\top$$

The magnitude of the epipolar line vector (excluding the homogeneous coordinate) is:
$$|\ell_{1,xy}|^2 = a_1^2 + b_1^2 \quad \text{and} \quad |\ell_{0,xy}|^2 = a_0^2 + b_0^2$$

---

## 2. Distance Formulas

### Symmetric Epipolar Distance (SED)
This is the metric implemented in the `sym_epipolar_distance` function. It calculates the squared Euclidean distance from the points to their respective lines in both images and sums them.

**If `squared=True`:**
$$d_{\text{SED}}^2 = e^2 \left( \frac{1}{a_1^2 + b_1^2} + \frac{1}{a_0^2 + b_0^2} \right)$$

**If `squared=False` (Average Point-to-Line):**
$$d_{\text{SED}} = \frac{1}{2} \left( \frac{|e|}{\sqrt{a_1^2 + b_1^2}} + \frac{|e|}{\sqrt{a_0^2 + b_0^2}} \right)$$



### Sampson Distance (First-Order Approximation)
The Sampson distance is a first-order approximation of the squared reprojection error. Unlike the SED, it uses a **joint normalization** in a single denominator.

$$d_{\text{Sampson}}^2 = \frac{e^2}{a_1^2 + b_1^2 + a_0^2 + b_0^2}$$

---

## 3. Implementation Breakdown (`torch.einsum`)

To compute distances for $N$ points in $p_0$ and $M$ points in $p_1$ simultaneously:
`p1_E_p0 = torch.einsum("...mi,...ij,...nj->...nm", p1, E, p0)`

This computes the constraint for every possible pair $(n, m)$:
$$\text{out}[..., n, m] = \tilde{\mathbf{p}}_1^{(m)\top} \mathbf{E} \tilde{\mathbf{p}}_0^{(n)}$$

---

## 4. Numerical Example

**Setup:**
$$E = \begin{bmatrix} 0 & -1 & 0 \\ 1 & 0 & 0 \\ 0 & 0 & 0 \end{bmatrix}, \quad \tilde{p}_0 = \begin{bmatrix} 1 \\ 0 \\ 1 \end{bmatrix}, \quad \tilde{p}_1 = \begin{bmatrix} 0 \\ 1 \\ 1 \end{bmatrix}$$

1. **Algebraic Error:** $e = (0,1,1) \cdot (0,1,0)^\top = 1$
2. **Line Norms:**
   - $\ell_1 = (0, 1, 0) \implies a_1^2 + b_1^2 = 1$
   - $\ell_0 = (1, 0, 0) \implies a_0^2 + b_0^2 = 1$

**Results:**
| Metric | Formula | Value |
| :--- | :--- | :--- |
| **SED (Squared)** | $1^2 \cdot (\frac{1}{1} + \frac{1}{1})$ | **2.0** |
| **Sampson** | $\frac{1^2}{1 + 1}$ | **0.5** |
| **Average Distance** | $\frac{1}{2} (1 + 1)$ | **1.0** |

---

## 5. Summary Note
The function `sym_epipolar_distance` computes the **Symmetric Epipolar Distance**. While often used interchangeably in casual conversation, the **Sampson approximation** is specifically the version with the combined denominator ($d = e^2 / \sum \nabla f^2$), which provides a tighter approximation to the true geometric reprojection error.


## 6. Math demo

### Derivation of the Mean Symmetric Epipolar Distance (SED)

The Symmetric Epipolar Distance (SED) measures the geometric error between a point and its corresponding epipolar line. In its non-squared form, it is typically expressed as the **average** perpendicular distance from the points to their respective epipolar lines in both images:

$$
d_{\text{SED}} = \frac{1}{2} \left( \text{dist}(p_1, \ell_1) + \text{dist}(p_0, \ell_0) \right)
$$
$$
d_{\text{SED}} = \frac{1}{2} \left( d_{0AB} + d_{0BA} \right)
$$

#### 1. General Distance Formula
The perpendicular distance $d$ from a 2D point $(x, y)$ to a line defined by $ax + by + c = 0$ is:

$$
d = \frac{|ax + by + c|}{\sqrt{a^2 + b^2}}
$$

#### 2. Deriving the Distance in Image 1
In the second image (Image 1), the epipolar line $\ell_1$ is generated by the point from the first image ($p_0$) using the Essential matrix $E$:

$$
\ell_1 = E p_0 = \begin{bmatrix} a_1 \\ b_1 \\ c_1 \end{bmatrix}
$$

The point we are measuring against is $p_1 = (x_1, y_1, 1)$.

* **The Numerator:** The dot product $\ell_1 \cdot p_1$ results in the algebraic error $e$. We take the absolute value for distance:
    $$|e| = |p_1^\top E p_0|$$
* **The Denominator:** The normalization factor is the magnitude of the line's normal vector $(a_1, b_1)$:
    $$\sqrt{a_1^2 + b_1^2}$$

Thus, the distance in Image 1 is:
$$\text{dist}(p_1, \ell_1) = \frac{|e|}{\sqrt{a_1^2 + b_1^2}}$$
