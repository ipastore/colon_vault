  - With ($T_{w\to c} = [R\ |\ t]$), a world point ($X_w$) is transformed as:
    
    $X_c = R X_w + t$
    
  - Depth in camera frame is the third component:
    
    $Z_c = (R X_w + t)_z = r_3^\top X_w + t_z$

## Demonstration of row vs coloumns
  $x_c = R x_w + t$
  

  Now define row form:  
  $\tilde{x}_w = x_w^\top,\quad \tilde{x}_c = x_c^\top,\quad \tilde{t}=t^\top$

  Transpose both sides:
  $\tilde{x}_c = (R x_w + t)^\top   = x_w^\top R^\top + t^\top   = \tilde{x}_w R^\top + \tilde{t}$
  
  For (N) points stacked as rows $(X_w\in\mathbb{R}^{N\times3}$):
  $X_c = X_w R^\top + \mathbf{1}, \tilde{t}$ 
  (in NumPy, broadcasting does the ($\mathbf{1},\tilde{t}$) part).

  Depth is the third camera coordinate:
  $z_c = (X_c)_{:,2}= (X_w R^\top){:,2} + t_z$
